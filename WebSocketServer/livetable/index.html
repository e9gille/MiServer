Databound tables

In this example we explore how changes to objects on the server can be pushed to the client using server initiated communication.

To try this out, go to the back end server session and set the variables in the #.Databinding namespace. Try things like:

  • #.Databinding.TableA←10 10(?⍴)100
  • #.Databinding.TableA[5;]←0
  • #.Databinding.TableA[;3]←99
